# Modern Intranet - based on 7.3 Formed Theme

A variation of the 7.3 Formed Theme (https://github.com/lfrsales/73-formed-theme) but optimized for Modern Intranets

### Set Up

1. `npm install`
2. `npm run init`
3. `npm run deploy`
